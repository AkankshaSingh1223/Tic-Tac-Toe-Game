The Tic Tac Toe Game is a desktop application developed using Java and Swing that recreates the classic two-player strategy game. The project provides an intuitive graphical user interface where players take turns placing X and O on a 3×3 grid. The application automatically detects winning combinations, draw conditions, and manages game flow efficiently.

This project demonstrates the practical application of Object-Oriented Programming (OOP) principles, event-driven programming, and GUI development in Java while emphasizing clean code structure and user-friendly design.

Key Features
Interactive 3×3 game board
Two-player gameplay
Automatic win and draw detection
Real-time game status updates
Restart/New Game functionality
Responsive graphical user interface
Clean and modular Object-Oriented design
